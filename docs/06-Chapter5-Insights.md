## 4. 交互逻辑

### 4.1 卡片展开/收起

| 交互元素 | 行为 | 效果 |
|----------|------|------|
| 认知卡片 | 点击 | 展开显示完整内容，再次点击收起 |
| 展开动画 | 高度过渡 | duration: 400ms, ease-out |
| 默认状态 | 初始 | 全部收起，仅显示标题 |

### 4.2 动画行为

| 交互元素 | 触发条件 | 行为 | 参数 |
|----------|----------|------|------|
| 章节标题 | 滚动进入视口 | 淡入 + 向上滑入 | duration: 600ms |
| 认知卡片 | 滚动进入视口 | 依次从下向上滑入 | stagger: 150ms |
| 卡片展开 | 点击 | 高度展开 + 内容淡入 | duration: 400ms |
| 卡片 hover | 鼠标悬停 | 轻微上浮 + 阴影 | translateY: -4px |
| 编号 | 滚动进入视口 | 数字滚动效果 | duration: 1000ms |

### 8.2 关键代码片段

**卡片展开/收起**
```tsx
import { useState } from 'react';
import { motion, AnimatePresence } from 'framer-motion';

const [expandedId, setExpandedId] = useState<number | null>(null);

<motion.div
  layout
  onClick={() => setExpandedId(expandedId === card.id ? null : card.id)}
  className="cursor-pointer"
>
  <div className="flex items-center gap-4">
    <div className="text-4xl font-bold text-blue-600">{card.number}</div>
    <div className="flex-1">
      <h3 className="text-lg font-semibold">{card.title}</h3>
    </div>
    <motion.div
      animate={{ rotate: expandedId === card.id ? 180 : 0 }}
      transition={{ duration: 0.3 }}
    >
      ▼
    </motion.div>
  </div>

  <AnimatePresence>
    {expandedId === card.id && (
      <motion.div
        initial={{ height: 0, opacity: 0 }}
        animate={{ height: 'auto', opacity: 1 }}
        exit={{ height: 0, opacity: 0 }}
        transition={{ duration: 0.4 }}
      >
        <div className="mt-4 pt-4 border-t">
          <p className="text-slate-600 leading-relaxed">{card.content}</p>
        </div>
      </motion.div>
    )}
  </AnimatePresence>
</motion.div>
```

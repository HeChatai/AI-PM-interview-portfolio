## 1. 模块概述

| 项目 | 内容 |
|------|------|

## 4. 交互逻辑

### 4.1 动画行为

| 交互元素 | 触发条件 | 行为 | 参数 |
|----------|----------|------|------|
| 主标题 | 页面加载 | 从下向上淡入 | duration: 500ms, delay: 0ms |
| 副标题 | 页面加载 | 从下向上淡入 | duration: 500ms, delay: 200ms |
| 数据卡片 | 页面加载 | 依次从下向上滑入 | duration: 600ms, delay: 400ms + index*100ms |
| 滚动引导 | 持续 | 上下浮动 | duration: 1000ms, repeat: infinite |
| 导航栏 | 滚动 >100px | 从顶部滑入固定 | duration: 300ms |
| 数据卡片 | hover | 上浮 + 阴影加深 | translateY: -4px, shadow: lg |

### 4.2 用户操作

| 操作 | 目标元素 | 效果 |
|------|----------|------|
| 点击 | 滚动引导 | 平滑滚动至下一模块（Chapter 1） |
| 点击 | 数据卡片 | （可选）弹出详细说明 Modal |
| 滚动 | 页面向下 | 导航栏滑入显示 |

## 7. 响应式设计

### 7.1 断点定义

| 断点 | 名称 | 宽度范围 | 布局调整 |
|------|------|----------|----------|
| `sm` | 小屏（移动） | ≥640px | 单列布局 |
| `md` | 中屏（平板） | ≥768px | 双列数据卡片 |
| `lg` | 大屏（桌面） | ≥1024px | 三列数据卡片 |

### 7.2 布局调整详情

**桌面端（≥1024px）**
- 主标题：56px
- 数据卡片：3 列横向排列
- 模块高度：100vh（满屏）

**平板端（768-1023px）**
- 主标题：42px
- 数据卡片：2 列排列（最后一个居中跨列）
- 模块高度：auto

**移动端（<768px）**
- 主标题：36px
- 数据卡片：单列垂直堆叠
- 模块高度：auto
- 滚动引导：固定在底部

## 8. 技术实现建议

### 8.1 组件结构

```
src/
├── components/
│   └── Hero/
│       ├── Hero.tsx          # 主组件
│       ├── HeroTagline.tsx   # 定位标签
│       ├── HeroTitle.tsx     # 主标题
│       ├── HeroSubtitle.tsx  # 副标题
│       ├── MetricCard.tsx    # 数据卡片
│       └── ScrollIndicator.tsx # 滚动引导
│
└── data/
    └── content.ts            # 数据配置
```

### 8.2 依赖库

| 库 | 用途 | 安装命令 |
|----|------|----------|
| `framer-motion` | 动画效果 | `npm install framer-motion` |
| `lucide-react` | 图标 | `npm install lucide-react` |
| `tailwindcss` | 样式 | `npm install -D tailwindcss` |

### 8.3 关键代码片段

**入场动画（Framer Motion）**
```tsx
import { motion } from 'framer-motion';

<motion.h1
  initial={{ opacity: 0, y: 20 }}
  animate={{ opacity: 1, y: 0 }}
  transition={{ duration: 0.5 }}
>
  从辅助到替代，我都做了什么
</motion.h1>
```

**数据卡片依次动画**
```tsx
const containerVariants = {
  hidden: { opacity: 0 },
  show: {
    opacity: 1,
    transition: {
      staggerChildren: 0.1
    }
  }
};

const itemVariants = {
  hidden: { opacity: 0, y: 20 },
  show: { opacity: 1, y: 0 }
};
```

## 9. 验收标准

### 9.1 功能验收

- [ ] 页面加载时所有动画按顺序执行
- [ ] 数据卡片 hover 有视觉反馈
- [ ] 滚动引导点击平滑滚动至下一模块
- [ ] 导航栏在滚动 >100px 后显示

### 9.2 视觉验收

- [ ] 配色与设计稿一致
- [ ] 字体大小、字重符合规范
- [ ] 间距符合设计规范
- [ ] 背景渐变和纹理效果正确

### 9.3 响应式验收

- [ ] 桌面端（1920x1080）显示正常
- [ ] 平板端（1024x768）显示正常
- [ ] 移动端（375x667）显示正常
- [ ] 所有断点下内容无溢出、无重叠

### 9.4 性能验收

- [ ] 首屏加载时间 < 2 秒
- [ ] 动画帧率 ≥ 60fps
- [ ] Lighthouse 性能得分 ≥ 90

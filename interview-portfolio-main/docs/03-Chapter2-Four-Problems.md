## 4. 交互逻辑

### 4.1 卡片展开/收起

| 交互元素 | 行为 | 效果 |
|----------|------|------|
| 问题卡片 | 点击 | 展开显示详情，再次点击收起 |
| 展开动画 | 高度过渡 | duration: 300ms, ease-out |
| 图标变化 | 展开时 | 箭头从 ↓ 变为 ↑ |
| 其他卡片 | 展开一个时 | 自动收起其他已展开的卡片（手风琴效果） |

### 4.2 动画行为

| 交互元素 | 触发条件 | 行为 | 参数 |
|----------|----------|------|------|
| 章节标题 | 滚动进入视口 | 淡入 + 向上滑入 | duration: 600ms |
| 问题卡片 | 滚动进入视口 | 依次从下向上滑入 | stagger: 100ms |
| 卡片展开 | 点击 | 高度展开 + 内容淡入 | duration: 300ms |
| 卡片 hover | 鼠标悬停 | 轻微上浮 + 阴影 | translateY: -4px |

import { useState } from 'react';
import { motion, AnimatePresence } from 'framer-motion';

const [expandedId, setExpandedId] = useState<number | null>(null);

const toggleCard = (id: number) => {
  setExpandedId(expandedId === id ? null : id);
};

<motion.div
  layout
  initial={false}
  animate={{ height: expandedId === card.id ? 'auto' : '120px' }}

## 4. 交互逻辑

### 4.1 动画行为

| 交互元素 | 触发条件 | 行为 | 参数 |
|----------|----------|------|------|
| 章节标题 | 滚动进入视口 | 淡入 + 向上滑入 | duration: 600ms |
| 对比卡片 | 滚动进入视口 | 左右同时滑入 | duration: 800ms |
| 核心洞察 | 对比卡片动画后 | 从下向上淡入 | delay: 400ms |
| 对比卡片 | hover | 轻微放大 + 阴影 | scale: 1.02 |

### 4.2 滚动触发

- 使用 **Intersection Observer** 检测模块进入视口
- 动画在模块完全进入视口后触发
- 支持重复触发动画（可选）

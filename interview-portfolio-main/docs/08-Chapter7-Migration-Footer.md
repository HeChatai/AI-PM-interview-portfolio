## 4. 交互逻辑

### 4.1 卡片交互

| 交互元素 | 行为 | 效果 |
|----------|------|------|
| 场景卡片 | hover | 轻微上浮 + 阴影 |
| 条件卡片 | hover | 轻微上浮 + 阴影 |
| 不适用卡片 | hover | 轻微上浮 + 阴影 |

### 4.2 Footer 交互

| 交互元素 | 行为 | 效果 |
|----------|------|------|
| 邮箱链接 | 点击 | 打开邮件客户端 |
| LinkedIn | 点击 | 跳转 LinkedIn 页面 |
| 下载按钮 | 点击 | 下载 PDF 文件 |
| 回到顶部 | 点击 | 平滑滚动到页面顶部 |

### 4.3 动画行为

| 交互元素 | 触发条件 | 行为 | 参数 |
|----------|----------|------|------|
| 章节标题 | 滚动进入视口 | 淡入 + 向上滑入 | duration: 600ms |
| 场景卡片 | 滚动进入视口 | 依次从下向上滑入 | stagger: 100ms |
| Footer | 滚动进入视口 | 淡入 | duration: 800ms |
| 回到顶部按钮 | 滚动 >500px | 从右下角淡入 | duration: 300ms |

## 7. 响应式设计

### 7.1 布局调整

| 断点 | 布局调整 |
|------|----------|
| `≥1024px` (桌面) | 适用场景和迁移条件左右并排（3 列 x2 行） |
| `768-1023px` (平板) | 适用场景和迁移条件上下堆叠 |
| `<768px` (移动) | 所有卡片单列垂直堆叠 |
